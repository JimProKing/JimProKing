# JimProKing

- 꾸준히 개발을 하며 내용들을 형상관리 하고 있습니다.
- Python, Java, dart(flutter) 등의 다양한 언어를 사용하고 있습니다.
  - java의 경우, 현재 직장에서 Spring 및 jsp를 사용하기에 매일 접하지만, 혼자 하는 개발에서는 사용하지 않습니다.

# 앱 개발 
- 10여개 모바일 앱을 혼자 제작 및 배포했던 경험이 있습니다.
- DB로는 supabase를 사용하였고, flutter 기반으로 제작하였습니다.

# 전산직 공무원
### 환경부 - 낙동강유역환경청
- 기간: 2024.10. ~ 2025.06.08.
- 업무: 국가하천 CCTV 모니터링 시스템 구축
* 인사교류로 국세청으로 전입할 기회가 생겨, 국세청 정보화관리관실로 근무지를 바꾸게 되었습니다.
### 국세청(본청) - 정보화관리관실
- 기간: 2025.06.09 ~ 현재
- 업무: 홈택스 및 손택스 전산시스템 고도화 및 운영관리
    - 손택스 밒 홈택스 포털 관련 업무
    - Java Spring 소스코드 형상관리 및 변경관리
    - 업무팀(각 세목 부서) 요청 및 세법 변경 관련 전산시스템 반영
